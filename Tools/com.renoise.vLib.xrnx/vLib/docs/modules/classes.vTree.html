Module classes.vTree

Tree view, used for displaying hierarchical data

Functions

update () update table using current settings (rebuild if needed)
update_tree (t, depth) recursive method, will iterate through table and update each row
get_visible_row_count (t) get the number of rows in the data which can be shown (used for determining range of the scrollbar)
parse_data (t) + add a unique id to all nodes, and cache this (.map) + initialize the "expanded" attribute
