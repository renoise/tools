Module Duplex.Device

The Device class is the base class for any OSC or MIDI device.

It also contains methods for managing basic device settings

Functions

__init (name, message_stream, protocol) Initialize the Device class
get_protocol () Retrieve the protocol of this device
set_control_map (xml_file) Set the device to the provided control-map
point_to_value () Convert a display/canvas point to an actual value, ready for output (always overridden with device-specific implementation, as the device will use MIDI or OSC-specific features at this stage)
